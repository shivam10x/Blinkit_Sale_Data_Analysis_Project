# Blinkit_Sale_Data_Analysis_Project


## 📌 Project Overview

This project analyzes sales data from Blinkit (a quick-commerce platform) to uncover insights about product performance, customer preferences, and outlet characteristics.

The goal is to use data analysis techniques to identify patterns and help improve business decisions.

---

## 📂 Dataset

* Dataset Name: Blinkit Grocery Dataset
* Source: Kaggle
* Contains information about:

  * Item Type
  * Item Visibility
  * Item Fat Content
  * Outlet Size
  * Outlet Location Type
  * Outlet Type
  * Sales

---

## 🎯 Objectives

* Analyze sales distribution across categories
* Identify high-revenue product types
* Study impact of outlet size on sales
* Understand relationship between visibility and sales
* Explore pricing vs sales trends

---

## ❓ Key Questions Answered

### 1. What are the most sold product categories?

 * Fruits, vegetables, and snacks are usually the most sold

### 2. Which category generates the highest revenue?

* Packaged foods and beverages contribute the most revenue.

### 3. How does outlet size affect sales?

* Medium and large outlets generate higher sales than small outlets.

### 4. Does product visibility impact sales?

* Higher visibility generally leads to higher sales.

### 5. Which outlet type performs best?

* Supermarkets usually perform best

### 6. Which outlet type performs best?

* Low-fat and regular items both perform well

### 7 . What is the average sales per outlet?

* Varies by outlet size and location
* Large outlets generally have higher averages
